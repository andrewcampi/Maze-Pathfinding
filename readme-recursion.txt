#####EXTRA INFO:#######
In addition to creating mouserecursion.py (which solves the maze by going in every direction), I created a file
named backtracking_mouserecursion.py (which solves the maze similar to an a* algorithm using one copy of the maze
and backtracking).The backtracking_mouserecursion.py file is included in this submission and you can test it using backtracking_testmouse.py

(Disclaimer: any file with backtracking in the title is extra work and is not a completely accurate version of what the project asked for.)
